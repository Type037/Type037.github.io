卡牌游戏项目

项目介绍

在这个项目中，我负责设计和实现卡牌系统。我设计了多种类型的卡牌，包括攻击、防御和技能卡牌，并实现了卡牌的组合效果，增加了游戏的策略性。同时，我开发了卡组管理功能，让玩家可以自由选择、添加或移除卡牌，并实时更新卡组状态。在 UI 交互界面方面，我设计并实现了主菜单界面，提供新游戏、加载游戏、设置等选项，界面简洁明了；战斗界面展示玩家手牌、敌人信息和战斗日志，支持拖放操作，增强了玩家的操作体验；卡牌库界面允许玩家浏览和选择可用卡牌，支持卡牌过滤和搜索功能；设置界面则允许玩家调整音量、难度等游戏参数。在数据管理方面，我使用 Json 格式存储卡牌信息和游戏进度，方便扩展和维护；通过 Excel 管理静态数据，便于平衡调整和内容更新。此外，我还实现了一个单例的 GameManager 来控制游戏状态、管理回合进程和事件触发。为了提升游戏的视觉和听觉体验，我使用 DOTween 系统和粒子特效增强战斗表现，集成音效库添加卡牌使用、敌人攻击等音效。

In this project, I was responsible for designing and implementing the card system. I designed various types of cards, including attack, defense, and skill cards, and implemented card combination effects to increase the game's strategic depth. Meanwhile, I developed the deck management function, allowing players to freely select, add, or remove cards and update the deck status in real - time. Regarding the UI interaction interface, I designed and implemented the main menu interface, which provides options such as new game, load game, and settings, with a simple and clear design. The battle interface displays the player's hand cards, enemy information, and battle logs, and supports drag - and - drop operations to enhance the player's operating experience. The card library interface allows players to browse and select available cards, supporting card filtering and search functions. The settings interface allows players to adjust game parameters such as volume and difficulty. In terms of data management, I used the Json format to store card information and game progress for easy expansion and maintenance, and managed static data through Excel for convenient balance adjustment and content updates. Additionally, I implemented a singleton GameManager to control the game state, manage the turn process, and trigger events. To enhance the game's visual and auditory experience, I used the DOTween system and particle effects to enhance the battle performance and integrated a sound effects library to add sound effects for card use, enemy attacks, etc.
